# 🎵 Spotify Clone

This is a simple Spotify Clone built with **React** to emulate core Spotify features like playing music, browsing playlists, and searching for songs. 🎧

## ✨ Features
- 🔑 User Authentication
- 📃 Browse Playlists
- 🎶 Play Music
- 🔍 Search Songs, Artists, and Albums
- 📱 Responsive Design

## 🛠️ Technologies Used
- ⚛️ React
- 🎨 CSS/Tailwind
